================
ImperviousRunoff
================

Simple runoff model multiplying the area with the rain.


Parameter
---------

+-----------------------+------------------------+------------------------------------------------------------------------+
|        Name           |          Type          |       Description                                                      |
+=======================+========================+========================================================================+
| area                  | double                 | impervious area in m2                                                  |
+-----------------------+------------------------+------------------------------------------------------------------------+

In and Out
----------

+--------------------+------------+--------------------------------------------+
|        Name        | Type       |   Description                              |
+====================+============+============================================+
| rain_in            | in         | rain fall in m                             |
+--------------------+------------+--------------------------------------------+
| out_sw             | out        | storm water run off                        |
+--------------------+------------+--------------------------------------------+
