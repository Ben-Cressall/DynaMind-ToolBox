===========
Merge Faces
===========

Merge faces off same attributes

Parameter
---------

+-------------------+------------------------+------------------------------------------------------------------------+
|        Name       |          Type          |       Description                                                      |
+===================+========================+========================================================================+
|view_name          | STRING                 | view name of the input faces                                           |
+-------------------+------------------------+------------------------------------------------------------------------+
|combined_view_name | STRING                 | view name of combined faces                                            |
+-------------------+------------------------+------------------------------------------------------------------------+
|attribute_name     | STRING                 | attribute name                                                         |
+-------------------+------------------------+------------------------------------------------------------------------+



Data-stream
-----------

+---------------------+--------------------------+------------------+-------+------------------------------------------+
|        View         |          Attribute       |       Type       |Access |    Description                           |
+=====================+==========================+==================+=======+==========================================+
| view_name           |                          | FACE             | read  | input faces                              |
+---------------------+--------------------------+------------------+-------+------------------------------------------+
|                     | attribute_name           | STRING           |       |                                          |
+---------------------+--------------------------+------------------+-------+------------------------------------------+
| combined_view_name  |                          | FACE             | write | resulting faces                          |
+---------------------+--------------------------+------------------+-------+------------------------------------------+


Module Name
-----------

:index:`GDALMergeFaces <GDALModules; GDALMergeFaces | Merge faces>`
