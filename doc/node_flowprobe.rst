==========
Flow Probe
==========

:index:`Sewer <CD3Node; FlowProbe | Measure Flow>` measurers the flow as state variable.
The node can be insert between two nodes to measure the flow without effecting the simulation.

State Variable
--------------

+-----------------------+------------------------+------------------------------------------------------------------------+
|        Name           |          Type          |       Description                                                      |
+=======================+========================+========================================================================+
| TotalFlow             | Double                 |                                                                        |
+-----------------------+------------------------+------------------------------------------------------------------------+
| Flow                  | Double Array           |                                                                        |
+-----------------------+------------------------+------------------------------------------------------------------------+
